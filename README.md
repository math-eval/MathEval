# MathEval

MathEval是一个专注于全面评估大模型数学能力的测评基准。共包含22个数学领域测评集和超过70K道数学题目，旨在全面评估大模型在包含算术，小初高竞赛和部分高等数学分支在内的各阶段、难度和数学子领域的解题能力表现，既可以作为现阶段大模型之间数学能力横向对比的一站式参考，也可以为后续如何进一步提高大模型数学能力指引方向。
